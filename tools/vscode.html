        # 为什么使用VS Code

        • 开源，免费，跨平台支持
        • 微软出品，质量保障
        • 扩展丰富，安装方便
        • 好用, 使用人群十分广泛

        # 插件安装

        直接快捷键 control + shift + x （windows）, command + shift + x (Mac) 下，调出应用商店。 在应用商店中搜索相关插件名字，点击安装，重启编辑器，即可使用。
