    # 前端微服务

    子应用互相独立，独立部署，独立编码。

    # Single-SPA

    single-spa是前端微服务框架。用于构建可共存的微前端应用，每个前端应用都可以用自己的框架编写，完美支持 Vue React Angular。

    • 在同一页面上使用多个前端框架 而不用刷新页面 (React, AngularJS, Angular, Ember, 你正在使用的框架)
    • 独立部署每一个单页面应用
    • 新功能使用新框架，旧的单页应用不用重写可以共存
    • 改善初始加载时间，迟加载代码

    # qiankun乾坤

    qiankun是基于single-spa的前端微服务框架实现。

    • 基于single-spa，开箱即用的API
    • 任意技术都能介入
    • 样式隔离
    • 资源预加载
