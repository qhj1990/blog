# 脚本配置文件

常常会在项目中看到一些配置文件，脚本文件。例如 .sh, .yml, Makerfile

# 1.YAML

YAML, Yet Another Markup Language，用于编写配置文件。

# 语法特点

  • 大小写敏感
  • 缩进表示层级
  • 空格缩进
  • 相同层级的元素左侧对齐

  • yaml vscode扩展，全面的YAML语言支持。支持JSON Schema 7及以下版本。

# 2.Makefile

大型项目通常会有很多源文件需要同时编译，不同功能模块的文件放在不同的文件夹下。Makefile决定编译哪些文件，编译的顺序，是否需要重新编译。

# 规则

//target: 目标文件
//prerequisites: 依赖文件
//command: 执行命令
