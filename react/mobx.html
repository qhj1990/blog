      # MobX-react

      简单可扩展的状态管理。使用mobx-react必须同时安装mobx

      # 资源

      • 官网
      • 中文文档

      # 装饰器

      # 重点概念

      # observable

      可观察的状态

      # computed

      创建计算值，expression 不应该有任何副作用而只是返回一个值。 如果任何 expression 中使用的 observable 发生改变，它都会自动地重新计算，但前提是计算值被某些 reaction 使用了

      # reactions

      而是会产生一些副作用，比如打印到控制台、网络请

      # action

      动作是任何用来修改状态的东西

      # autorun

      当观察数据发生改变时，就会调用 autorun 的回调函数。

    const component = children || render

    }
    return useObserver(component)
