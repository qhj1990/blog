      # React-Router

      React Router是React的路由解决方案

      • github github仓库代码
      • 官网

      # 代码结构

      React Router5代码采用lerna和yarn workspace部署的monorepo。有多个packages。

      • react-router 核心代码
      • react-router-config 配置静态路由
      • react-router-dom 基于react-router，扩展了功能Link、HashRouter、BrowserRouter等
      • react-router-native react-native的路由实现

      # 基础

      react-router采用history这个库来处理路由的变化。

      # API

      # hooks

      利用React.useContext

      • useHistory
      • useLocation
      • useParams
      • useRouteMatch

      # Router

      所有路由器组件的通用底层接口。

      • <BrowserRouter> HTML5 history API
      • <HashRouter> hash
      • <MemoryRouter> url的history不会在地址栏上读取和更改
      • <NativeRouter>
      • <StaticRouter> url永不改变。 服务端渲染的时候

      # Route

      最基本的职责是在其路径与当前URL匹配时呈现某些UI
