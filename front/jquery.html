      # jQuery.extend(object)

      为jQuery类添加类方法，可以理解为添加静态方法

      jQuery.extend({
    min: function(a, b) { return a < b ? a : b; },
    max: function(a, b) { return a > b ? a : b; }
});
jQuery.min(2,3); //  2
jQuery.max(4,5); //  5
